# Minecraft Server

[SpigotMC](https://www.spigotmc.org)

## Getting started

```bash
sh host_setup.sh
sh server_setup.sh
sh server_build.sh
sh server_config.sh
sh server_start.sh
```

### Maintenance

* Backup: `server_backup.sh`
* Reset: `server_reset.sh`
* Restore: `server_restore.sh`
